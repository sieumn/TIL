# 개발 스터디

> 개발 스터디를 진행하며 매주 배우고 느낀 점을 기록합니다. <br>
> 현재 진행형으로, 매주 업데이트 됩니다.

## Fi-Pi

### 스터디 개요

- 주식 자동매매 프로그램 개발 스터디
- 기간: 주 1회 미팅 (시작일: 2021.09.14)

### 스터디를 통해 얻고싶은 것

- 설계부터 개발 및 테스트까지 전체 개발 과정 경험
- 클린 아키텍처를 적용한 소프트웨어 구현
- 매주 코드 리뷰를 통한 코드 작성 습관 개선
- 수익이 발생하는 알고리즘 만들기

### 언어 및 스킬

- Language: Python
- `스터디 진행하며 업데이트 예정`
